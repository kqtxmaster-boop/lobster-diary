小话梅版 OpenClaw - 无需翻墙，全程中文

🌟 为什么选择小话梅版？

  • 📦 国内镜像源 - 无需翻墙即可安装
  • 🇨🇳 全程中文 - 告别英文界面
  • ⚡ 配置简单 - 输入 API Key 就能用
  • 💬 多渠道支持 - 飞书、企业微信、Telegram
  • 🌐 Web 控制台 - 浏览器直接聊天

📋 安装步骤

  1. 点击下方按钮下载安装脚本
  2. 双击运行 install.sh（Mac/Linux）或 install.ps1（Windows）
  3. 按照提示输入 API Key
  4. 开始使用！
⚠️ 安装前请先准备好 API Key（推荐 MiniMax）
