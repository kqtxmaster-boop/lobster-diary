📖 使用文档

🚀 快速开始

1. 一键安装

复制以下命令到终端运行：

curl -fsSL https://www.xiaohuamei.my/install.sh | bash

2. 输入 API Key

安装过程中会提示您输入 API Key，您可以选择：

  • MiniMax (推荐) - 国内首选，稳定便宜
  • Kimi - 月之暗面出品
  • SiliconFlow - 聚合多家模型

3. 启动服务

安装完成后会自动启动，或者运行：

openclaw gateway --port 18789

🌐 如何使用

方式一：网页版

打开浏览器访问：http://localhost:18789/

方式二：飞书

  1. 打开 飞书开放平台
  2. 创建企业自建应用
  3. 获取 App ID 和 App Secret
  4. 添加权限：im:message:send_as_bot, im:chat:create
  5. 运行 openclaw channels login feishu 配置

方式三：企业微信

  1. 登录企业微信管理后台
  2. 创建自建应用
  3. 获取 AgentId 和 Secret
  4. 运行 openclaw channels login wecom 配置

💰 获取 API Key

MiniMax (推荐)

  1. 访问 MiniMax 开放平台
  2. 注册账号并实名认证
  3. 在"API Keys"页面创建密钥
  4. 复制密钥备用

SiliconFlow

  1. 访问 SiliconFlow
  2. 注册账号
  3. 在控制台获取 API Key
💡 提示：新用户通常有免费额度，可以先用免费额度测试

⚙️ 常用命令

# 启动服务
openclaw gateway

# 查看状态
openclaw status

# 配置聊天渠道
openclaw onboard

# 查看帮助
openclaw --help

🔧 常见问题

Q: 安装失败怎么办？

确保已安装 Node.js 22+，如果没有会自动安装。请检查网络连接。

Q: 一直显示"正在启动"？

按 Ctrl+C 停止，然后重新运行 openclaw gateway

Q: 如何更换 API Key？

编辑 ~/.openclaw/openclaw.json 文件，修改对应的 API Key

Q: 如何后台运行？

# Linux/Mac
nohup openclaw gateway &

# 使用 systemctl
systemctl --user start openclaw-gateway
⚠️ 注意：OpenClaw 需要运行在有网络的环境下。如果无法访问国外 API，请使用国内模型（MiniMax、Kimi 等）
