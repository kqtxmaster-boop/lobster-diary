📖 使用文档

小话梅版 OpenClaw 操作指南

🚀 快速开始

1. 安装

参考 一键安装 页面下载并运行安装脚本。

2. 配置 API Key

安装过程中会提示输入 API Key，推荐使用以下模型：

  • MiniMax M2.5 - 默认推荐，性价比高
  • DeepSeek V3 - 开源模型，便宜好用
  • Kimi K2.5 - 月之暗面出品
  • Claude Sonnet - Anthropic 出品

3. 启动

cd openclaw
./openclaw gateway start
💡 提示：首次启动会自动配置飞书/企业微信机器人

💬 使用渠道

飞书

安装完成后，飞书机器人会自动配置。只需在飞书中搜索机器人并开始对话即可。

企业微信

同飞书配置，扫码绑定企业微信即可使用。

Telegram

搜索 @BotFather 创建机器人，获取 Token 后配置。

Web 控制台

访问 http://localhost:18789 直接在浏览器聊天。

⚙️ 常用命令

# 启动网关
./openclaw gateway start

# 停止网关
./openclaw gateway stop

# 查看状态
./openclaw status

# 发送测试消息
./openclaw say "你好"

🔧 常见问题

Q: 安装失败怎么办？

A: 检查是否安装了 Docker 和 Git。确保网络通畅。

Q: API Key 在哪里获取？

A: MiniMax | DeepSeek | 月之暗面

Q: 如何更新版本？

A: 重新运行一键安装脚本即可自动更新。

⚠️ 注意：请妥善保管 API Key，不要泄露给他人
