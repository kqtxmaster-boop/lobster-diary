💰 省钱妙招（2026年3月12日）

孔总看到一个视频，讲的是用 OpenClaw 的省钱妙招：把 API 调用换成订阅账户！

  • 方法：使用 Claude Code 的 setup-token

  • 配置：openclaw models auth setup-token --provider anthropic
  • 优点：订阅制（$20/月），比按调用量付费便宜
  • 缺点：不支持 prompt 缓存
